# Log & Help Files Archive

이 폴더는 프로젝트 개발 및 설정 과정에서 생성된 임시 기록 파일들을 포함하고 있습니다. 루트 디렉토리를 깔끔하게 유지하기 위해 이곳으로 이동되었습니다.

## 파일 구성 및 목적

### 1. 도움말 파일 (`*_help.txt`)
- **대상**: `engine_help.txt`, `inferencer_help.txt`, `folder_help.txt` 등
- **목적**: Anomalib 라이브러리의 엔진, 추론기, 폴더 구조 관련 CLI 명령어 옵션들을 확인하기 위해 생성된 파일입니다. 개발 중 참고용으로 사용되었습니다.

### 2. 실행 로그 (`log*.txt`)
- **대상**: `log.txt`, `log2.txt`, `download_log.txt` 등
- **목적**: 데이터 준비, 모델 학습, 라이브러리 설치 등 주요 프로세스 실행 시 터미널에 출력된 내용을 기록한 파일입니다. 문제 발생 시 추적을 위해 보관되었습니다.

### 3. UTF-8 변환 파일 (`*_utf8.txt`)
- **대상**: `*_utf8.txt`
- **목적**: Windows 환경에서 한글이 포함된 로그 파일이 깨지는 것을 방지하기 위해 UTF-8 인코딩으로 변환된 복사본입니다.

---
> **참고**: 이 파일들은 실제 시스템 실행에는 영향을 주지 않으므로, 더 이상 필요하지 않다고 판단되면 폴더째로 삭제하셔도 무방합니다.
